# ❤️ Heart Disease Classification - End-to-End Machine Learning Project

## 📌 Overview

This project is an **end-to-end machine learning pipeline** that predicts whether a person has heart disease based on medical attributes. It uses supervised learning techniques and follows a complete workflow from data preprocessing to model evaluation.

The goal of this project is to demonstrate practical implementation of machine learning concepts using real-world healthcare data.

---

## 🚀 Features

* Data loading and exploration using **Pandas**
* Data visualization using **Matplotlib** and **Seaborn**
* Data preprocessing and feature analysis
* Model training using multiple algorithms
* Hyperparameter tuning using **GridSearchCV** and **RandomizedSearchCV**
* Model evaluation using multiple performance metrics
* Comparison of different models to select the best one

---

## 🛠️ Tech Stack

* **Python**
* **NumPy**
* **Pandas**
* **Matplotlib**
* **Seaborn**
* **Scikit-learn**

---

## 📊 Dataset

The dataset used in this project contains medical attributes such as:

* Age
* Sex
* Chest pain type
* Resting blood pressure
* Cholesterol levels
* Maximum heart rate achieved
* And more...

The target variable indicates:

* `0` → No heart disease
* `1` → Presence of heart disease

---

## 🔍 Machine Learning Workflow

### 1. Data Exploration & Analysis

* Understanding data distribution
* Identifying correlations between features
* Visualizing patterns using plots

### 2. Data Preprocessing

* Handling missing values (if any)
* Feature selection and transformation
* Splitting data into training and testing sets

### 3. Model Building

The following models were trained and compared:

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Random Forest Classifier

### 4. Model Evaluation

Models were evaluated using:

* Accuracy Score
* Precision
* Recall
* F1 Score
* Confusion Matrix
* ROC Curve

### 5. Hyperparameter Tuning

* Used **GridSearchCV** and **RandomizedSearchCV** to optimize model performance
